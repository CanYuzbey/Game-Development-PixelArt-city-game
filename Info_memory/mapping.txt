MAP BUILDER ENGINE - C++ TECHNICAL REFERENCE

Project: Isometric Pixel Art Open World City RPG
Current source model: C++17

OVERVIEW

The mapping system generates a complete city district map from a single integer
seed. The same seed and configuration must produce the same map structure every
time.

Core native types:

- mapping_algorithm::MapConfig
- mapping_algorithm::MapCell
- mapping_algorithm::MapGrid
- mapping_algorithm::MapStats
- mapping_algorithm::MapGenerator
- mapping_algorithm::DesignBlueprint

GAME INTEGRATION

Include:

  #include "mapping_algorithm/map_generator.hpp"

Create a config, construct MapGenerator, call generate(), then read grid(),
stats(), and to_design_blueprint().
